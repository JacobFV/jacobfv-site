## Overview

SQTEST is a rule-based English – Spanish translation system written in C# with Visual Studio. It supports past and present tense translation in both directions and formalizes many aspects of the English – Spanish mapping (verb conjugation, gender/number agreement, common idiomatic substitutions).

Source code lives at [JacobFV/SQT-English-to-Spanish-Translator](https://github.com/JacobFV/SQT-English-to-Spanish-Translator).
